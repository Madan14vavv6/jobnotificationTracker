## Spacing system

- **Scale**
  - `xs` = 4px
  - `sm` = 8px
  - `md` = 12px
  - `lg` = 16px
  - `xl` = 24px
  - `2xl` = 32px
  - `3xl` = 40px

- **Page layout**
  - Horizontal padding: `xl` on desktop, `lg` on tablet, `md` on small screens.
  - Vertical padding around main content: `xl` on desktop, `lg` on smaller screens.
  - Gap between primary workspace and secondary panel: `xl`.

- **Cards and surfaces**
  - Internal padding: `xl` (desktop), `lg` (mobile).
  - Vertical spacing between sections inside a card: `lg`.

- **Forms**
  - Vertical spacing between fields: `md`.
  - Spacing between field label and control: `xs`.
  - Spacing between control and helper/error text: `xs`.

- **Headers**
  - Top spacing above primary content header: `xl` (desktop), `lg` (mobile).
  - Spacing between header and workspace content: `xl` (desktop), `lg` (mobile).
